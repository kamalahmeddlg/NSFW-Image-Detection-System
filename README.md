# NSFW Image Detection System

An AI-based deep learning system for detecting NSFW (Not Safe For Work) images using CNN, EfficientNet and Explainable AI (Grad-CAM).

## Features
- Detect NSFW and Safe images
- Deep learning based classification
- Grad-CAM explainability
- Flask web interface
- Model training pipeline

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Flask
- Matplotlib
